Tillbaka till lista över taggar

Description

Innehåller en beskrivning av sidan. Sökmotorer kan plocka upp denna beskrivning och visa den som ett resultat vid sökningar

Exempel

<!DOCTYPE html>
<html>

<head>
<meta name="description" content="En beskrivning av "description" i name attributet">
</head>

<body>
</body>

</html>
