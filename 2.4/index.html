Bilder och bakgrunder

Background

Egenskapen background används för att sätta samtliga bakgrunds egenskaper i en och samma deklaration.

De egenskaper som går att sätta i samband med denna är följande.

  • background-color
  • background-image
  • background-position
  • background-size
  • background-repeat
  • background-origin
  • background-clip
  • background-attachment

Ifall man inte vill ha med någon av de egenskaperna som listas ovan så kan man bara hoppa över den och deklarera nästa istället.

Exempel

div { background: #00ff00 url("bgPic.png") no-repeat fixed center; }

img och css

För att kombinera bilder och text på ett snyggt sätt kan man i CSS använda sig av en egenskaper som heter float och clear tillsammans med HTML taggen <img> och övriga text taggar så som <p> eller headingtaggar. Float bestämmer om elementet ska vara "flytande" eller ej samt vilket håll det i så fall ska "flyta". Att ha ett "flytande" element innehär att elementet som kommer efter kommer att lägga sig brevid, antingen till höger eller vänster. Så om jag väljer att en <img> ska ha float: left; så kommer bilden att lägga sig till vänster på skärmen och den <p>-tagg som kommer efter kommer lägga sig brevid till höger.

Clear

Men för att kunna bestämma att element inte längre ska "flyta" på sidan om ett annat element längre så kan man använda sig av egenskapen clear. Denna egenskap berättar för webbläsaren att elementet inte ska flyta runt det andra elementet längre och lägger det istället nedanför i detta fall bilden.

Exempel

#imgCss img {
margin: 0 20px 10px 10px;
float: left;
}

#imgCss h3{
clear:left;
}
