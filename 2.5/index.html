Länktillstånd

En länk i ett HTML dokument kan hanteras på olika sätt beroende på vilket tillstånde den är i. Till exempel så kan den se ut på ett sätt ifall användaren har besökt länken tidigare, eller ifall de drar muspekaren över dem.

Detta görs via CSS genom att lägga till en så kallad pseudoklasser till sin selektor. Med hjälp av ett kolon efter selektorn och namnet på pseudoklassen kan man välja att styla ett visst tillstånd på ett specielt sätt. Till exempel ifall jag skulle vilja att länken ska vara gul när användaren trycker på den så kan jag skriva enligt följande.

a:active {
color: yellow;
}

Eller ifall jag vill att länken ska bli större när användaren drar sin mus över den så kan jag skriva enligt följande.

a:hover {
font-size: 2em;
}

Pseudo-klasser

  • a:link
    • Pseudo-klassen "link" används när man vill göra förändringar på en länk som ännu inte är besökt av användaren.
  • a:active
    • Pseudo-klassen "active" används när man vill göra förändringar på en länk vid tillfället som användaren klickar på den.
  • a:visited
    • Pseudo-klassen "visited" används när man vill göra förändringar på en länk som användaren har besökt tidigare.
  • a:hover
    • Pseudo-klassen "hover" används när man vill göra förändringar på en länk som användaren håller sin muspekare över.

Cursor

Egenskapen "cursor" används för att specificera hur en muspekare ska se ut när användaren håller den över ett visst element. För att se en mer specifik lista över alla typer av muspekare man kan välja på, gå till w3schools
