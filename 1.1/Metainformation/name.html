Name-attribut

Name-attributet specificerar ett namn till din metadata så att det kan hanteras på ett korrekt sätt. Du kan till exempel lägga in en beskrivning av sidan, vem som har skapat sidan och andra nyckelord för sidan.

Exempel

<!DOCTYPE html>
<html>

<head>
<meta name="description" content="Free Web tutorials">
<meta name="keywords" content="HTML,CSS,JavaScript">
<meta name="author" content="Hege Refsnes">
</head>

<body>
</body>

</html>
