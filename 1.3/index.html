Semantiska element för text

<mark>

<mark>-taggen används för att definera markerad text. Denna lägger man in när man vill markera ett kort stycke av en text.

Exempel:

Glöm inte bort att köpa mjölk på vägen hem från jobbet.

<time> med datetime attribut

<time>-taggen används för att skriva in ett datum/tid som är läsbart av en människa. Taggen kan också användas för att lägga in datum och tid som kan läsas av en dator. Detta görs genom att lägga till datetime attribut till taggen. Genom att lägga in datum och/eller tid som kan läsas av en dator kan man på så sätt erbjuda sina användare att lägga till tidpunkten i sin kallender, det hjälper även sökmotorer att visa ett bättre resultat.

Exempel:

Vi öppnar butiken kl 10:00 varje morgon

Kod:

<time datetime="10:00">10:00</time>
