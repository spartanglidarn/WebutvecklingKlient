Webbläsar-objekt



Ett nytt objekt kommer läggas till i listan varje sekund.

Med hjälp av nedan knappar kan du starta och stoppa listan.

Om du klickar på nedan knapp så kommer en alert box komma upp efter 5 sekunder.

För att stoppa alert box från att visas efter 5 sek så kan ni klicka på nedan knapp
