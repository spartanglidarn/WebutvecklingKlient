Text och listor

Fontfamiljer

Font-family egenskapen specificerar fonten som ska användas i ett element på sidan.

Font-family egenskapen kan hålla flera olika fonter som fungerar som ett "fallback" system ifall webbläsaren inte skulle kunna hantera en viss font.

För att ändra storlek på texten kan man använda sig av två primära enheter. nyckelord(xx-small, x-small, small, medium, large, x-large och xx-large) eller värden(% och em.

Nyckelord

Nycklord sätter fontstorleken till ett fastväde som är definerat av webbläsaren som finns på användarens dator.

Värden

Värde är ett relativt värde som sätter fontens storlek utifrån det fontstorleken som är satt i förälder-elementet.

<head>-tag

Color

Egenskapen color bestämmer vilken färg en font ska ha. Man kan välja att använda sig av egna färger eller fördefinerade systemfärger.

Egna färger anger man genom att sätta ett värde med en så kallad "hexadecimal".

Fördefinerade systemfärger är färger som redan finns inlagda i HTML och det skrivs med klar text så som, red, green, black och så vidare. Se exempel nedan.

Exempel

p { color:red; } ul li { color: #111; }

Teckensnitt

Enheten font sätter alla font egenenskaper i en deklaration. De egenskaper som kan sättas i en font deklaration är följande font-style font-variant font-weight font-size/line-height font-family

De obligatoriska värdena i font enheten är font-family samt font-size. Om en av de andra värdena saknas så sätts de standardvärde som finns.

Exempel

p { font: italic bold 12px/30px Georgia, serif; }

Listor

Egenskapen list-style är en egenskap som sätter alla egenskaper list deklarationer i en och samma deklaration.

De egenskaper som kan sättas i list-style är följande list-style-type list-style-position list-style-image

Ifall något värde saknas i list-style så kommer det ett default värde sättas, om det finns ett.

Exempel

p { list-style: square inside; }
