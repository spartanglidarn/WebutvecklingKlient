Teckenkod

Teckenkodning läggs in högst upp i <head> taggen av ett HTML dokument för att berätta för webläsaren vilken kodning dokumentet har. Deklarationen ska rymmas inom de första 1024 bytes från början av filen.

Deklarationen kan läggas in på två olika sätt. Antingen genom att använda elementet meta med attributet charset. Eller genom att använda attributen http-equiv och content. Se nedan exempel

Exempel 1: meta charset

<!DOCTYPE html>
<html>

<head>

<meta charset="utf-8"/>
</head>

<body>
</body>

</html>

Exempel 2: http-equiv och content

<!DOCTYPE html>
<html>

<head>

<meta http-equiv="Content-Type" content="text/html; charset=utf-8"/>
</head>

<body>
</body>

</html>
