Tabeller och formulär

table-layout

Egenskapen table-layout används för att sätta en viss typ av layout algoritm som ska användas för tabellen. De värden som går att sätta till denna egenskap är följande.

  • auto

    Auto är det värde som ligger som default på en tabel ifall man inte sätter något alls. Kollumnens bredd sätts efter den längsta ej brytbara innehållet i en cell. Denna typ av rendering kan orsaka segare laddning då webläsaren behöver läsa igenom allt innehåll i en tabell innan den kan sätta en slutgiltig layout.

  • fixed

    Detta innebär att den horisontella designen endast är beroenda av tabellens bredd samt bredden av kollumnerna. Alltså inte innehållet i kollumnerna. Denna typ av design gör att webbläsaren kan rendera sidan lite snabbare då den kan börja visa tabellen så fort första raden har tagits emot.

  • initial

    Sätter värdet i table-layout till default värdet.

  • inherit

    Gör så att elementet ärver sitt värde från "förälder-elementet".

Exempel

table {
table-layout: auto;
}

Css med formulär

Stilen på ett formulär kan göras betydligt mycket bättre genom att använda sig av CSS. Något som är viktigt att komma ihåg när man designar ett formulär är att göra det så enkelt som möjligt för användaren att förstå vad som ska skrivas in i formuläret. Detta för att till så hög grad som möjligt undvika att felaktig data kommer in till systemet från användaren.

I själva stilsättningen användar man samma egenskaper och samma typ av selektorer som när man sylar andra objekt på en sida, till exempel tabeller. Något som kan vara väldigt bra att använda sig av dock är pseudo-klasserna för input type. Detta gör det möjligt att visa olika typer av rutor på olika sätt och på så sätt visa för användaren vilken typ av information som ska skrivas i input-fältet.

I nedan exempel visar vi hur man stylar alla input-fält som är av typen text med en svart ram som har rundade kanter, samt att vi har lagt till en bakgrundsfärg i fältet när användaren väljer att markera fältet.

Exempel

input[type=text] {
border: 1px solid grey;
border-radius: 15px;
}

input[type=text]:focus {
background-color: #3D9970;
}
