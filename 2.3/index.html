Marginaler, kantlinjer och paddning

Boxmodell

Man kan se alla HTML element i ett dokument som lådor. I CSS när man pratar om Boxmodellen så talar man om design och layout.

Boxmodellen i CSS är i grund och botten en som man lägger ett HTML element i. Lådan består av margin(ger ett avstånd utanför lådan som är genomskinligt), border(en kant som går runt innehållet och padding delen), padding(ger ett avstånd runt innehållet i lådan som är genomskinligt) och det faktiska innehållet i elementet.

Exempel

div { width: 300px; border: 25px solid green; padding: 25px; margin: 25px; }

Margin

Egenskapen margin sätter alla de margin egenskaper som finns i en deklaration. Så istället för att sätta fyra olika för varje sida av lådan så sätter man istället alla fyra värden i en och samma egenskap.

Om man ger egenskapen endast ett värde så blir det samma margin på alla fyra sidor. Om man ger egenskapen två eller tre värden så får de sidor utan värde ett värde av sin motsatta sida, till exempel om man skriver margin: 10px 5px; så kommer övre och undre sidan få 10px medans höger och vänster får 5px.

I nedan exempel så är blir margin-top 10px, margin-right 5px, margin-bottom 15px och margin-left 20px.

Exempel

div { margin:10px 5px 15px 20px; }

Border-color

Egenskapen border-color sätter en förg på den kant som går runt lådan. Egenskapen sätter färgen på alla fyra sidor av lådan och färgen kan anges i antingen fördefinerade systemfärger eller i hexadecimaler.

Om man vill ange färg för en specifik sida av lådan så kan man använda sig av egenskaperna border-top-color, border-right-color, border-bottom-color eller border-left-color.

Det går även att sätta olika färger på olika sidor av lådan genom att ange flera värden i egenskapen border-color. Exemplet nedan visar till exempel på att vi har sätt varje sida av lådan till olika färg så som denna artikels kanter ser ut.

Exempel

div { border-color: red green blue pink; }

Border-style

Egenskapen border-style bestämmer vilken stil kanten ska ha. Det går till exempel att välja ifall man vill ha en prickig kant eller en solid kant, eller ifall man inte vill ha någon kant alls.

Exempel

div { border-style: dotted; }

Border-width

Egenskapen border-width bestämmer tjockleken på en kant. Genom att ge egenskapen ett värde så kan man välja ifall det ska vara en tjock eller en smal kant. Detta kan göras genom att sätta värdet i fördefinerade systemstorlekar eller genom att sätta värdet i px(pixlar)

Exempel

div { border-style: solid; border-width: 15px; }

Padding

Egenskapen padding bestämmer avståndet mellan innehållet i taggen till kanten som går runt "lådan". Detta används för att generera avstånd runt om innehållet.

Nedan exempel ger 5px på varje sida om innehållet till kanten.

Exempel

div { padding: 50px; }

Outline

Egenskapen outline är en linje som skrivs ut runt om ett element. Utanför kanten(border) men innanför den marginal(margin) man sätter.

Detta används för att få en del av sidan att synas lite mer än andra på samma sätt som vi har gjort för denna artikel.

Exempel

div { outline: #00FF00 dotted thick; }

Visibility

Egenskapen visibility bestämmer ifall en del på sidan ska synas eller inte. Delar som med hjälp av visibility egenskapen inte visas på skärmen tar ändå upp plats. För att gömma delar som inte ska ta upp någon plats kan man använda sig av display egenskapen.

De vanligaste värden som används med visibility är hidden och visible.

Exempel

div { visibility: hidden; }

Visibility

Egenskapen visibility bestämmer ifall en del på sidan ska synas eller inte. Delar som med hjälp av visibility egenskapen inte visas på skärmen tar ändå upp plats. För att gömma delar som inte ska ta upp någon plats kan man använda sig av display egenskapen.

De vanligaste värden som används med visibility är hidden och visible. Men det finns även fler egenskaper som ni kan läsa mer om på w3chools.com

Exempel

div { visibility: hidden; }

clip

Egenskapen clip används för att klippa av ett element som har en absolut position ifall elementet blir större än det element som det ligger i.

Exempel

div { position: absolute; clip: rect(0px,60px,200px,0px); }

Display

Egenskapen display definerar vilken typ av "låda" som ska användas för HTML elementet. Detta kan användas på en rad olika sätt, till exempel om du sätter värdet "none" så kommer elementet att inte visas på skärmen och den kommer inte heller att påverka resten av designen. Använder du värdet "list-item" så kommer elementet bete sig som att den låg i en <li>-tagg. För att se fler värden att använda tillsammans med egenskapen display gå till W3shcools.com

Exempel

div { display: list-item; }

Overflow

Egenskapen Overflow specificerar vad som kommer hända ifall andra element skulle överskugga detta elements låda. Egenskapen bestämmer ifall innehållet i elementet ska klippas av eller ifall det ska läggas till en så kallad scrollbar om när ett elements innehåll är för stort för att få plats på en specifik yta.

Exempel

div { width: 150px; height: 150px; overflow: scroll; }

Transparant box

Det finns olika sätt att åstakomma en box som är gradvis transparant, här ska jag förklara två av dom.

RGBA

RGBA är ett värde och en förlängning på färgsättnings modellen RGB där "A" står för alpha som är ett värde som avgör dess transparenta förmåga. 0.0 är helt genomskinligt medans 1.0 är fullt färgsatt. Till exempel så skulle {background-color: rgba(255, 0, 0, 0.3);} innebära att bakgrundsfärgen blir en genomskinlig röd färg.

Opacity

Egenskapen Opacity sätter en nivå av genomskinlighet där 0.0 är helt genomskinligt och 1.0 är standardvärdet för enheten. Om man till exempel sätter {opacity: 0.5;} så blir elementet till 50% genomskinligt.

Exempel

div { opacity: 255,0,0,0.3; }

div2 { background-color: red; opacity: 0.3; }

Runda hörn

Egenskapen border-radius används för att lägga till rundade hörn på ett element. Ifall man endast anger ett värde i egenskapen så får alla fyra hörn samma värde.

Men man kan även välja att lägga in två, tre eller fyra värden till egenskapen.

Vid två värden så anger det första värdet det övre vänstra hörnet samt det nedre högra, och det andra värdet anger det övre högra samt nedre vänstra.

Vid tre värden så anger det första värdet det övre vänstra, det andra värdet anger det övre högra samt det nedre vänstra, och det tredje värdet anger det nedre högra hörnet.

Vid fyra värden så anger det första värdet det övre vänstra hörnet, det andra värdet anger det övre högra hörnet, det tredje värdet anger det nedre högra hörnet, och det fjärde värdet anger nedre vänstra hörnet.

Exempel

div { border: 2px solid; border-radius: 10px 15px 20px 25px; }

Multippla bakgrundsbilder

Genom att använda egenskapen background-image i CSS så kan man lägga till multippla bakgrundsbilder. Om man vill använda flera bilder så kan man separera dem med ett komma och de kommer då att ligga ovangå varandra där den första bilden är överst. Du kan sedan positionera de olika bilderna genom att använda egenskapen background-position.

Exempel

#div {
background-image: url(bgPic.gif), url(bgPic2.gif);
background-position: right bottom, left top;
background-repeat: no-repeat, repeat;
}

Skuggad box

För att sätta en skugga på en box kan man använda sig av egenskapen box-shadow. Den sätter en eller flera skuggor på ett HTML element.

Exempel

#div { box-shadow: 10px 10px 50px #888888; }

Gradienter

Gradienter i CSS3 gör så att du kan skapa mjuka övergångar mellan två eller fler färger.

Tidigare var man tvungen att använda bilder för att skapa denna typ av effekt. Men genom att använda CSS3 gradients så kan man öka laddhastigheten och bandbredden som behövs för att öppna sidan.

  • Linjär Gradient (går ner/upp/vänster/höger/diagonalt)
  • Radiär Gradient (definerat av sin centerpunkt)

Linjär gradient

För att skapa en linjär gradient så måste man definera minst två färgstopp. Färgstopp är färgerna som man vill ska rendera med en mjuk övergång. Man kan ockå sätta en start punkt och en riktning tillsammans med gradientens effekt. Se exempel nedan(denna box är gjord med en linjär gradient).

Radiär gradient

För att skapa en radiär gradient så måste du även här definera mist två färgstopp. Man kan här även sätta värde för form. Den första färgen defineras som start färg och kommer att vara den som sitter i centrum. Se exempel nedan.

Exempel

Linjär gradient

#div { box-shadow: 10px 10px 50px #888888; }

Radiär gradient

#div { background: radial-gradient(blue, green, red); }
