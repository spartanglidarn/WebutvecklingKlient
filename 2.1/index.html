Grunderna i CSS 3

Externt länkade CSS-fil

När du har extern CSS innebär det att du har en separat CSS-fil som innehåller den CSS-kod som hör till hemsidan. Detta kallas för en stilmall. Om du då vill göra en större förändring i designen på din hemsida så kan du gå in i stilmallen och göra dina förändringar.

Detta görs genom att man länkar sin CSS-fil till HTML-sidan och får på så sätt information om hur varje emlement och tagg ska visas. Länkkoden ska läggas in i <head>-taggen. Se exempel nedan

Exempel

<link rel="stylesheet type="text/css" href="style.css">

Arv

Många av de CSS-regler som man sätter upp går i arv till sina barn-element(ett barn-element är ett element som skrivs innuti ett annat element). Detta innebär att många av de regler som du sätter upp gäller för flera element även fast de är deklarerade till endast ett. Men det är bara vissa egenskaper i css som går i arv till barn-element. Till exempel så går "color" egenskapen i arv, men "border" egenskapen går inte i arv till sina barn-element.

Exempel

p { color:red; } article { border:1px solid black; }

Cascade

"Cascade" syftar på en algoritm som definerar hur man kombinerar egenskapers värde som kommer från olika källor. Detta ligger till grund för hela kärnan i CSS och det visas genom att C i CSS står för Cascade(Cascade style sheet).

Selektorer

En selektor talar om vad i HTML-dokumentet du vill formatera. Ofta består en selektor av HTML-taggens namn utan < och >. De kan även bestå av klasser eller identifierare som man sätter på taggarna i HTML-dokumentet. All information som ska vara kopplat till en selektor skriver man inom måsvingar( {} ).

Exempel

p { color:red; } article { border:1px solid black; }

Enheter

I CSS kan man sätta storlekar som absoluta eller relativa enheter.

En absolut enhet innebär att storlek inte påverkas av andra objekt i dokumentet utan visas endast i förhållande till vilket värde enheten har. Några aboslutat enheter följer nedan.

  • in (inches)
  • cm (centimeter)
  • mm (milimeter)
  • pt (points)
  • pc (picas)

En Relativ enhet innebär att storlek påverkas av andra eneheter i dokumentet. Relativa enheter skalar bättre mellan olika renderings medium, så som skärmstorlek, upplösning och pixelmängd. Några absoluta enheter följer nedan.

  • em (relativ till fontstorleken på elementet)
  • ex (relativ till x-höjden av den tillfälliga fonten, används sällan)
  • px (relativ till antalet pixlar skärmen har)
  • % (relecant till den satta default-storleken på enheten)

CSS-dokumentets uppbyggnad

CSS-kodens instruktioner som även kallas stilmallar består av regler som berättar för webbläsaren hur data ska representeras på den skärm det visas. En sådan regel består av fyra olika delar enligt nedan

  • Selektor(berättar vilken del av dokumentet som ska stylas)
  • Deklaration(sker inom måsvingar efter selektorn)
  • Egenskap (berättar vad det är som ska ändras inom selektorn)
  • Värde (berättar vilket värde en egenskap ska ha, till exempel storleken på en font)
