Tillbaka till lista över taggar

dl, dt och dd lista

<dl> står för description list och används tillsammans med <dt> och <dd>. <dl> definerar och omfamnar sjävla listan, du kan sedan lägga till namnet på ett listobjekt genom att lägga till <dt> som är namnet på listobjektet och slutligen en beskrivning av listobjektet genom att lägga till <dd>.

HTML
Ett märkspråk för hypertext och utgör tillsammans med TCP/IP och HTTP den grundläggande standarden för WWW (World Wide Web, webben).
CSS
Ett språk som beskriver presentationsstilen för ett strukturerat dokument som till exempel typsnitt, textstorlek och färg
JavaScript
Ett prototyp-baserat skriptspråk som är dynamiskt, svagt typat och hanterar funktioner som första-klass-objekt.

Exempel

<!DOCTYPE html>
<html>

<head>
</head>

<body>
<dl>
<dt>HTML</dt><dd>Ett märkspråk för hypertext och utgör tillsammans med TCP/IP och HTTP den grundläggande standarden för WWW (World Wide Web, webben).</dd>
<dt>CSS</dt><dd>Ett språk som beskriver presentationsstilen för ett strukturerat dokument som till exempel typsnitt, textstorlek och färg</dd>
<dt>JavaScript</dt><dd>Ett prototyp-baserat skriptspråk som är dynamiskt, svagt typat och hanterar funktioner som första-klass-objekt.</dd>
</dl>
 </body>

</html>
