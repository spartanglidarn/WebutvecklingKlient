http-equiv

http-equiv attributet ger en HTTP-header till den information om innehållet i attributet.

http-equiv attribut kan användas för att simulera ett HTTP svars header. I exemplet nedan uppdateras dokumentet var 30e sekund.

Exempel

<!DOCTYPE html>
<html>

<head>
<meta http-equiv="refresh" content="30">
</head>

<body>
</body>

</html>
