@Page selektor

@Page selektorn i CSS används för att specificera hur en sida är formaterad inom en rektagulär yta med fast bredd och höjd.

Denna selektor kan användas med flera psudo-klasser för att specificera utseenden på vissa specifika sidor så som till exempel höger och vänster, eller första sidan.

Nedan exempel sätter sidans storlek till att vara 7,5 inches bred och 15 inches hög. Detta är då det som kommer att visas på varje sida.

Exempel

@page {
size:7,5in 15in;
margin: 1cm;
}

page-break-after

Egenskapen page-break-after bestämmer ifall ett sidbryt ska ske efter ett visst element. Tillexempel ska man sätta denna egenskap på en footer för att vara säker på att sidbryt sker varje gång en footer har skrivits ut på sidan.

Exempel

footer{
page-break-after: always;
}

page-break-before

Egenskapen page-break-before bestämmer ifall ett sidbryt ska ske före ett visst element. Ett bra användaningsområde för detta kan vara att sätta egenskapen på alla H1 taggar så att sidbryt alltid sker innan man skriver ut ett H1 element

Exempel

h1 {
page-break-before: always;
}

page-break-inside

Egenskapen page-break-inside bestämmer ifall ett sidbryt är tillåtet innut i ett element. Detta kan till exempel användas på P taggar för att undvika säkerställa att ett sidbryt inte sker mitt i en paragraf.

Exempel

p {
page-break-inside: avoid;
}

Size

Värdet size används för att sätta storleken på en "page-box". Detta kan sättas som antingen relativt värde eller absolut värde.

Nedan exempel sätter sidans storlek till att vara 7,5 inches bred och 15 inches hög. Detta är då det som kommer att visas på varje sida.

Exempel

@page {
size:7,5in 15in;
margin: 1cm;
}

orphans

Egenskapen orphans sätter eller retunerar det minsta antalet rader för ett element som måste vara synbart på slutet av en sida. Detta gäller för skrivare och förhandsgranskning för skrivare.

Om vi till exempel sätter värdet 4 på orphans så betyder det att det måste visas minst 4 rader i slutet av en sida innan det blir sidbryt.

Exempel

p {
orphans: 5;
}

widows

Egenskapen widows sätter eller retunerar det minsta antalet rader för ett element som måste vara synbart i början av en sida. Detta gäller för skrivare och förhandsgranskning för skrivare.

Om vi till exempel sätter värdet 4 på orphans så betyder det att det måste visas minst 4 rader i början av en sida

Exempel

p {
widows: 5;
}
